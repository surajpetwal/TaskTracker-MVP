# Task Tracker - Privacy Policy

**Last Updated:** April 2026

## Overview
Task Tracker is a personal productivity app that helps you track daily tasks and build habits. This privacy policy explains how we handle your data.

## Data Collection

**What we collect:**
- Task data (titles, descriptions, completion status)
- Points and streak information
- App usage statistics (stored locally only)

**What we DON'T collect:**
- Personal information (name, email, phone)
- Location data
- Internet usage data
- Any data transmitted to external servers

## Data Storage

All data is stored **locally on your device** using SQLite database. No data is transmitted to any server or cloud service.

## Permissions

The app requires minimal permissions:
- **Storage:** To save task data locally
- **Notifications:** (Optional) For task reminders

## Third-Party Services

This app does not use any third-party analytics, advertising, or tracking services.

## Your Rights

Since all data is local:
- You can clear all data by uninstalling the app
- Data is never shared with anyone
- You have complete control over your information
